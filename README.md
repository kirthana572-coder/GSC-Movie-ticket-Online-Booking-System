# GSC Movie Ticket Online Booking System

## Project Description
This system is a web-based movie ticket booking system for Golden Screen Cinema (GSC). It allows customers to browse movies, select seats, and make bookings online with physical payment at the cinema counter. The system also supports staff and admin operations such as payment verification, ticket generation, and system management.

---

## Objectives
- Provide easy and user-friendly movie booking
- Reduce congestion at cinema counters
- Allow seat selection and booking tracking
- Support staff in verifying payments and managing bookings

---

## End Users
- Customer
- Staff
- Admin
- Manager

---

## System Features

### Customer
- Register account
- Sign in / Sign out
- Forgot password (before login)
- Change password (after login)
- View & edit profile
- Browse movies & showtimes
- Select cinema branch
- Choose date, time, and seat
- Make booking
- View booking history
- View booking details (Pending / Paid)
- Receive QR e-ticket (after payment)
- Cancel booking (if not paid)

---

### Staff
- Sign in / Sign out
- Forgot & change password
- View profile & edit profile
- View customer bookings
- Verify counter payment
- Update payment status (Pending → Paid)
- Generate / print ticket
- Scan QR ticket
- Manage walk-in booking (Add, Edit, Delete, View)

---

### Admin
- Sign in / Sign out
- Forgot & change password
- View profile
- Add / Edit / Delete / View movies
- Add / Edit / Delete / View showtimes
- Add / Edit / Delete / View seats
- View & delete users
- Add / Edit / Delete / View staff

---

## Payment Method
- Physical payment at cinema counter
- Booking created with status: **Pending**
- After payment, staff updates status to: **Paid**
- System generates QR E-Ticket after payment

---

## Main Flow (Customer)

1. Register / Login  
2. Browse movies  
3. Select cinema branch  
4. Choose date & time  
5. Select seats  
6. Confirm booking → Status: Pending  
7. Pay at counter  
8. Staff updates status → Paid  
9. Receive QR E-Ticket  
10. Scan ticket at cinema
